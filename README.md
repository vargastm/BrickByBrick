## ğŸ¯ Smart Contracts & Deployment



BrickByBrick uses smart contracts to manage tokenization, milestone tracking, and automated fund releases based on verified construction progress.



### What the contracts do



**Tokenization:** Projects are represented as ERC-20 tokens, enabling fractional ownership and trading.



**Milestone Escrow:** Funds are locked in smart contracts and released only after AI-validated construction milestones are completed on-chain.



**Progress Verification:** AI systems verify construction progress and update on-chain state, triggering automatic fund releases.



**Secondary Trading:** Tokens can be traded on secondary markets, providing liquidity for investors.



### Supported Chains



- Ethereum Mainnet

- Polygon

- Optimism

- Arbitrum

- Base

### Prerequisites



- Node.js 18+ and npm/yarn

- A Web3 wallet (MetaMask, Coinbase Wallet, or Rainbow)

- Environment variables configured (see `.env.example`)



### Installation



```bash

# Clone the repository

git clone https://github.com/letpires/BrickByBrick.git

cd BrickByBrick



# Install dependencies

npm install



# Set up environment variables

cp .env.example .env.local

# Edit .env.local with your configuration



# Run the development server

npm run dev

```



Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
